PROJECT // 01
mlx-nim
A local LLM/VLM inference server for Apple Silicon — no cloud, no telemetry, no data leaving your machine. Exposes three simultaneous API surfaces (OpenAI-compatible, Anthropic-compatible, and custom /api), making it a zero-modification drop-in replacement for cloud APIs. Supports streaming, tool calling, structured JSON output, vision inputs, KV cache quantization, speculative decoding, and reasoning blocks. Benchmarked at 2.4× faster prompt processing vs. standard Ollama (201 t/s vs. 85 t/s) and 2.2× faster tool-call generation (41 t/s vs. 19 t/s).
